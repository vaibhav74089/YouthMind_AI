  - Accessibility features (WCAG 2.1 compliant)

- **Documentation**

  - Code comments and documentation

#### Features by Category

**Emotion Detection**
- Anxious, Sad, Angry, Happy, Tired, Confused states
- Crisis keyword detection
- Default/neutral state handling

**Coping Strategies**
- 4-7-8 breathing technique
- 5-4-3-2-1 grounding method
- Journaling prompts
- Movement suggestions
- Mindfulness exercises
- Social connection recommendations

**Safety Features**
- Real-time crisis monitoring
- Emergency contact information
- Professional resource links
- Clear disclaimers about limitations

#### Technical Specifications
- **Backend**: Python 3.8+, Flask 3.0.0
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI**: Rule-based NLP with keyword matching
- **Storage**: Session-based (no persistent storage)
- **Security**: No personal data collection, GDPR compliant
